//--------------------步驟--------------------
另一種釋出 Angular 應用的簡單途徑是使用 GitHub Pages。
1. 你需要建立一個 GitHub 賬號（如果沒有的話），然後為你的專案建立一個倉庫。記下 GitHub 中的使用者名稱和專案名。
2. 使用 Angular CLI 命令 ng build 來建構這個 GitHub 專案，選項如下：
ng build --prod --output-path docs --base-href /<project_name>/
3. 當建構完成時，把 docs/index.html 複製為 docs/404.html。
4. 提交你的更改，並推送。
5. 在 GitHub 的專案頁中，把該專案配置為從 docs 目錄下發布。
你可以到 https://<user_name>.github.io/<project_name>/ 中檢視部署好的頁面。
